# proyecto-equipo-PAR

## 📌 Descripción
Aplicación de consola desarrollada en .NET para la gestión de tareas.  
Permite crear, listar, completar, eliminar tareas y filtrar, facilitando la organización y seguimiento de actividades.

## 💻 Tecnologías utilizadas
- .NET 9 (SDK)  
- Aplicación de consola (CLI)  

## ⚙️ Requisitos
Antes de ejecutar el proyecto, asegurarse de tener instalado:

- .NET 9 SDK  
  - La instalación se realiza mediante asistente (wizard)  
  - No requiere configuración adicional de variables de entorno  

## ▶️ Ejecución del proyecto

Ejecutar los siguientes comandos:

```bash
git clone https://github.com/RodrigoET/proyecto-equipo-PAR.git
cd proyecto-equipo-PAR
cd TaskApp
dotnet run
```

📌 Una vez ejecutado, se iniciará la aplicación en modo consola, permitiendo interactuar con sus funcionalidades.

## 🧩 Funcionalidades principales
- ➕ Agregar tareas  
- 📋 Listar tareas  
- ✅ Completar tareas  
- ❌ Eliminar tareas  
- 🔍 Filtrar por categoría  
- ⚡ Filtrar por prioridad
